ReadMe for the Buttons Tutorial

About this Tutorial

This tutorial demonstrates the use of PIO lines as an input signal in several small and progressive steps. This tutorial assumes that you have a basic knowledge of the C language and have read and understood Writing BlueCore Applications User Guide (CS-110344-UG).

This tutorial is divided into four parts:

  1. Basic PIO messages
  2. Button handlers 1: Pressing a button, releasing a button, and double-clicking a button
  3. Button handlers 2: Holding a button, holding and then releasing a button, and repeated messages whilst holding down a button
  4. Button handlers 3: Advanced button handling and further tasks

Each part is wrapped in a separate xIDE workspace (partn.xiw) and contains the source code and ReadMe for a simple (in structure and functionality) and working example application.

To start this tutorial, open part1 workspace (tutorials/buttons/part1.xiw) in xIDE.

Additional Notes

  1. This top-level workspace and project is intentionally empty.
  2. The tutorials contain code fragments from the provided source code files.
  3. A source code line prefixed by the '+' symbol means that it is a required code change, whereas a source code line prefixed with the '-' symbol means that the code should be removed.
