ReadMe for the Blinking Lights Tutorial

About this Tutorial

This tutorial demonstrates the use of PIO lines as output for LEDs, starting with a simple single blinking LED and progressing to a sophisticated multi-pattern LED display using auto-generated code. This tutorial assumes basic knowledge of the C language.

This tutorial is divided into three parts:

  1. Blinking Lights 1: Using Message Handlers and timed messages to make LEDs blink
  2. Blinking Lights 2: The API for controlling dedicated LED supported by some BlueCore variants
  3. Blinking Lights 3: Using a .led configuration file to auto-generate LED pattern display code

Each part is wrapped in a separate xIDE workspace (bl_partn.xiw) and contains the source code and ReadMe for a simple (in structure and functionality) and working example application.

To start this tutorial, open bl_part1 workspace (tutorials/blinking_lights/bl_part1.xiw) in xIDE.

Additional Notes

  1. This top-level workspace and project is intentionally empty.
  2. The tutorials contain code fragments from the provided source code files.
  3. Part 1 requires devices with PIO-driven LEDs.
  4. Part 2 requires devices with onboard (non-PIO driven) LED support.
  5. Part 3 requires devices with either PIO-driven LEDs or onboard (non-PIO driven) LED support (e.g. BlueCore5 modules).
